# Ad Generation Monacaサンプル

### 概要

- このプロジェクトは、ブラウザIDEにてMonaca提供のサンプルに広告の表示を確認するプロジェクトになります。
- TOPページの表示確認及び画面遷移後の広告表示を確認をすることが出来ます。

### 動作確認環境
 - Monacaプラグイン3.0.0
 - OnsenUI 2
 - Cordovaバージョン
   - CLIバージョン5.2.0
   - iOSプラットフォーム3.9.1
   - Androidプラットフォーム4.1.1

### 注意事項
 - 広告枠IDは必ずテスト用のIDから変更をして下さい。
